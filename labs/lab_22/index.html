Lab 22: Reading and Writing Files

Due Friday, May 08 at 11:59 PM

Goals for This Lab

By the time you have completed this work, you should be able to:

  • Create File objects
  • Open files for reading with Scanner
  • Read the contents of files using Scanner
  • Use finally to make an action occur whether or not an exception is thrown
  • Write to files with BufferedWriter and FileWriter
Provided files:
  • PrintEveryOtherLine.java
  • colors.txt
  • WriteArgsToFile.java
  • collaborators.txt
Rubric(100 points):
  • PrintEveryOtherLine.java: 50
  • WriteArgsToFile.java: 50

Step-by-Step Instructions


Step 1: Download and Edit PrintEveryOtherLine.java

This program will open a given file for reading, specified on the command-line. It will then print out every other line of the file, printing the first line of the file.

For example, if you first download colors.txt, and then run your program with the command-line argument colors.txt, you should get the following output: 

1: Red
3: Green
5: Orange
7: Black
9: Pink

The comments in the program provide more details.


Step 2: Download and Edit WriteArgsToFile.java

This program will take some command-line arguments, and write them to a new file named “args.txt”, one per line. For example, if you run your program with the command-line arguments foo bar baz, it should produce a file named args.txt with the following contents: 

foo
bar
baz

The comments in WriteArgsToFile.java provide further details.

Note that the code you write in writeArgsToFile should write to whatever file is passed to writeArgsToFile. In the given program, this is always args.txt, provided by main. However, this could be any arbitrary file, and I will test your code with alternative File objects.
